# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, **do not open a public issue**.

Instead, email the maintainers directly or open a [private security advisory](https://github.com/sellomakgatho121/Fx-analyzer/security/advisories/new).

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## What to Expect

- Acknowledgment within 48 hours
- Regular updates on progress
- Credit upon disclosure (if desired)

## Responsible Disclosure

We request a 90-day window from notification to public disclosure, allowing time for a fix.
